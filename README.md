# 🎾 Sweet Spot Therapy

What if the most satisfying moment in tennis became a tiny interaction game?

Sweet Spot Therapy is an experimental web interaction inspired by the feeling of hitting the sweet spot on a tennis racket.

Instead of simulating real tennis, it recreates the anticipation, timing, and satisfying feedback of a clean hit through playful interactions, sound, and personality-based results.

## Features

- 🎾 Pull back and swing to hit the ball
- 💥 Different hit quality feedback
- 🗣️ Funny coach commentary
- 🧠 Tennis personality result
- 📱 Mobile friendly

## Built with

- React
- Vite
- ChatGPT
- Codex
